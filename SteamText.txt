A simple mod that grants Archotech bionics special max HP bonuses to make them even more viable.

Package ID: V1024.EliteArchotech

[h1]Changes Made[/h1]
[list]
[*]Archotech Arm: x110%, +15 max HP to installed body part
[*]Archotech Leg: x110%, +15 max HP to installed body part
[/list]

This mod uses the Elite Bionics Framework to achieve the max HP changing effect.

The code of this mod is available on GitHub [url=https://github.com/Vectorial1024/EliteArchotech]here[/url] as an example of how to use the Elite Bionics Framework.
